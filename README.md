# Task Manager

## О проекте

Task Manager — тестовое приложение для [NDA]

## Стек

### Core

- React 19
- TypeScript 5.9
- Vite 7
- Redux Toolkit + RTK Query
- React Router DOM 6.30

### UI & Forms

- MUI 6
- React Hook Form + Zod

### Testing

- Vitest
- Storybook
- Playwright

### Tools & CI/CD

- ESLint + Prettier
- Husky + lint-staged
- GitHub Actions
- json-server (Mock API)

## Требования

- Node.js >= 22.12.0 и доступна в PATH
- npm >= 10.9.0 и доступен в PATH
- Git установлен и доступен в PATH

## Запуск проекта

```bash
git clone <repository-url>
cd task-manager
npm install
npm run dev
```
